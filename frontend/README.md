# GROZA Frontend

React application for GROZA - Funeral Services Platform wireframes/POC.

## Quick Start

```bash
npm install
npm run dev
```

The app will open at http://localhost:3000

## Demo Credentials

- **Email:** Any valid email
- **Password:** `grozademo!`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Features

- ✅ Login page with demo authentication
- ✅ Role-aware dashboard
- ✅ Responsive design
- ✅ Clean, modern UI

## Tech Stack

- React 18
- Vite
- React Router DOM
